AlternativeX

AlternativeX is Angular UI & Component library concept created by me. Yea it's concept for now, if I find energy I will create full lib from this. LGPL licenced uwu.
AlternativeX is content focused, light and blazingly fast library. Yea I know, it's built for Angular, but hey! Angular can be blazingly fast aswell if not loaded with tons of shit.
AlternativeX is act of rebellion against Web 2.0, loaded with tons and tons of javascript, css, heavy images, heavy ads, tracking shit. It's all so heavy that Facebook{{' '}} for example is able to use more resources than fucking video game, ya know 3D with graphics and textures. It's sick.
I want my pages to be clean, fast, reactive and responsive. The lighter and simpler UI is, the more responsive experience you can get out of it.

And I don't know, I feel kinda charmed by sites that looks so simple, so basic, yet so clean, so I decided I need one for myself ❤️
